    (static) ClientLayout(props) → {JSX.Element}

    A React layout component that waits until client-side hydration is complete before rendering children. Displays a splash screen on first load for 3 seconds, and avoids flickering or mismatches during SSR hydration. Uses sessionStorage to remember if the splash screen has already been shown in the current session.

    (static) useHydrated() → {boolean}

    Custom React hook that determines if the component has hydrated on the client side. Sets `hydrated` to true after the first client-side render, allowing conditional rendering based on hydration status (e.g., to avoid mismatches between server and client rendering).
